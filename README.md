# Arduino Learning
Practice code from Paul McWhorter's Arduino tutorial series.
Topics covered:
- LED control (single and multiple)
- Digital output (Binary)
- PWM (Brightness, LED dimming)
- Push button input
- Toggle switch logic
- Active buzzer
- Analog read (ADC)
- Potentiometer
- Photoresistor (LDR)
- String input via Serial
- While loops / control flow
- Servo Motor Control
- Mathematical mapping
- Mini-Project: Weather Monitor (LDR + Servo + PWM LED0
